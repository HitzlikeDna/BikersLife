Anleitungen & Hilfe

Wie erstelle ich eine Gruppe?

Starte deine eigene Tour und lade Freunde ein. So geht's:

  1. Öffne die BikersLife App und tippe auf dem Startbildschirm auf das große "+" Symbol.
  2. Wähle "Neue Gruppe erstellen" aus.
  3. Gib deiner Gruppe optional einen Namen (z.B. "Sonntagsfahrt").
  4. Die Gruppe ist nun aktiv! Du siehst sofort einen QR-Code und einen 6-stelligen Gruppencode auf deinem Bildschirm.
  5. Zeige den QR-Code deinen Freunden zum Scannen oder teile den Code über WhatsApp/Messenger.

Wie trete ich einer Gruppe bei?

Verbinde dich schnell mit einer bestehenden Crew:

  1. Öffne die App und tippe auf "Beitreten".
  2. Du hast nun zwei Möglichkeiten:

    A) QR-Code Scannen: Die Kamera öffnet sich. Halte sie einfach über den Code des Gruppenführers.
    B) Code eingeben: Tippe den 6-stelligen Code ein, den du erhalten hast.
  3. Sobald erkannt, bist du automatisch im Voice-Chat verbunden.
  4. Stecke das Handy weg und fahre los – die Verbindung bleibt im Hintergrund aktiv.

Audio & Navigation

BikersLife ist darauf ausgelegt, parallel zu laufen.

  • Starte zuerst deine Musik oder Google Maps Navigation.
  • Öffne dann BikersLife und trete der Gruppe bei.
  • Die Lautstärke der Navigation wird nicht reduziert (kein "Ducking"), damit du Anweisungen klar verstehst, während du mit deinen Freunden redest.
